# Recursos visuales

Los recursos SVG incluidos en `public/assets/images` fueron creados específicamente para este prototipo. No contienen logos de fabricantes, modelos de producto, interfaces reales, especificaciones técnicas ni fotografías de operaciones.

La presentación técnica disponible se utilizó únicamente para comprender categorías generales de capacidad. No se copiaron imágenes de producto porque pueden contener marcas, referencias o información técnico-comercial que no debe publicarse.

El wordmark de Eagle Commercial incluido es **provisional** y debe reemplazarse por el archivo oficial aprobado por la compañía antes de publicar.
